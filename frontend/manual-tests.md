# Pruebas manuales de navegación con teclado

1. Abrir la aplicación y asegurarse de que la barra superior sea visible.
2. Usar la tecla `Tab` hasta que el foco esté en la pestaña activa.
3. Presionar las flechas izquierda y derecha para mover el foco entre las pestañas.
4. Verificar que el contenido cambia y que la pestaña seleccionada tiene `aria-selected="true"`.
5. Presionar `Escape` para cerrar el menú si está abierto.
